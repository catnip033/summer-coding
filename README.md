# 규칙

**1. 풀이한 코드는 회차 내부에 각자 폴더를 만든후, 자신의 폴더에 저장해주세요.**
```
1주차 > 1단계 > 김유신 > BOJ1000.cpp
```

코드 상단에 문제 링크 주소 넣어주세요.

```js
/*
    설탕 배달
    https://www.acmicpc.net/problem/2839
*/
```


# 깃허브에 파일 올리는 법
1. github 들어가서 Invite a collaborator로 초대 > 초대받은 사람이 자기 이름 폴더 만들고 코드 넣기
2. Git으로 이용하는 방법
 Git 다운로드 후, 파일을 저장하고 싶은 위치에 가서, 우클릭 후에 Git Bash Here 실행.
 
 다음 코드를 따라 작성
 
 ```
 $ git clone https://github.com/catnip033/summer-coding.git
 ```
 
 그러면 summer-coding 이라는 파일이 생성되므로   
 그 폴더에서 주차 > 단계 > 자기이름 폴더 생성 하고 코드 올리면 됨.   
 그 과정은 아래 참고하고   
 하다가 막히면 구글링
